# Verifying installation of OpenShift Container Platform clusters

If the OpenShift Container Platform cluster is in the installing state for over 30 minutes and has not become ready, ensure the AWS account environment is prepared for the required cluster configurations. If the AWS account environment is prepared for the required cluster configurations correctly, try to delete and recreate the cluster. If the problem persists, contact support.

.Procedure

- Verify the AWS account environment is prepared for the required cluster configurations.
- If the AWS account environment is prepared correctly, try to delete and recreate the cluster.
- If the problem persists, contact support.
