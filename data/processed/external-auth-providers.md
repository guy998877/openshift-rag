# Direct authentication identity providers

Direct authentication has been tested with the following OpenID Connect (OIDC) identity providers:

- Active Directory Federation Services for Windows Server
- GitLab
- Google
- Keycloak
- Microsoft Entra ID
- Okta
- Ping Identity
- Red Hat Single Sign-On

> **NOTE:** Red Hat does not test all factors associated with third-party identity provider functionality. For more information about third-party support, see the link:https://access.redhat.com/third-party-software-support[Red Hat third-party support policy].
