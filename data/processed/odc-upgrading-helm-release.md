# Upgrading a Helm release

You can upgrade a Helm release to upgrade to a new chart version or update your release configuration.

.Procedure

1. In the *Topology* view, select the Helm release to see the side panel.
1. Click *Actions* -> *Upgrade Helm Release*.
1. In the *Upgrade Helm Release* page, select the *Chart Version* you want to upgrade to, and then click *Upgrade* to create another Helm release. The *Helm Releases* page displays the two revisions.
