# Searching the Red Hat Knowledgebase

In the event of an OpenShift Container Platform issue, you can perform an initial search to determine if a solution already exists within the Red Hat Knowledgebase.

.Prerequisites

- You have a Red Hat Customer Portal account.

.Procedure

1. Log in to the link:http://access.redhat.com[Red Hat Customer Portal].

1. Click *Search*.

1. In the search field, input keywords and strings relating to the problem, including:
- OpenShift Container Platform components (such as *etcd*)
- Related procedure (such as *installation*)
- Warnings, error messages, and other outputs related to explicit failures

1. Click the *Enter* key.

1. Optional: Select the *OpenShift Container Platform* product filter.

1. Optional: Select the *Documentation* content type filter.
